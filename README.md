# 🎮 Mon Projet Unity

## 📌 Description
Ceci est un jeu développé avec Unity.  
Un jeu de plateforme où le joueur doit éviter des obstacles et atteindre la fin du niveau.

## 🚀 Fonctionnalités
- Gestion des déplacements du joueur
- Système de score
- Menu principal et options
